LOOI idle movement lab

Idle scenarios before we wire them in.

Tune small autonomous body motions for the moments when LOOI is silent or only talking. This demo edits wheel-level steps, converts them to linear/angular commands, and can send them to ESP32 when connected and armed.

Mobile note: keep the phone/robot lifted or wheels off the table for the first tests.
